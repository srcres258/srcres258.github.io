泰勒展开式 & 等价无穷小

泰勒展开式

每天起床头件事, 先背一遍展开式.

$ sin x = x - (1)/(6)x^3 + o(x^3) $

$ arcsin x = x + (1)/(6)x^3 + o(x^3) $

$ tan x = x + (1)/(3)x^3 + o(x^3) $

$ arctan x = x - (1)/(3)x^3 + o(x^3) $

$ ln (1 + x) = x - (1)/(2)x^2 + (1)/(3)x^3 + o(x^3) $

$ upright(e)^x = 1 + x + (1)/(2)x^2 + (1)/(6)x^3 + o(x^3) $

$ cos x = 1 - (1)/(2)x^2 + (1)/(24)x^4 + o(x^4) $

$ (1 + x)^alpha = 1 + alpha x + (alpha (alpha - 1))/(2)x^2 + o(x^2) $

$ a^x - 1 tilde.equive x ln a $

积分公式

$ integral x^mu , \mathrm{d}x = (1)/(mu+1)x^{mu+1}+C \quad (mu != -1) $

$ integral (1)/(sqrt(x)) , \mathrm{d}x = 2sqrt(x) + C $

$ integral (\mathrm{d}f(x))/(sqrt(f(x))) = 2sqrt(f(x)) + C $

$ integral upright(e)^x , \mathrm{d}x = upright(e)^x + C $

$ integral a^x , \mathrm{d}x = (a^x)/(ln a) + C $

$ integral (1)/(x) , \mathrm{d}x = ln |x| + C $

$ integral sin x , \mathrm{d}x = -cos x + C $

$ integral cos x , \mathrm{d}x = sin x + C $

$ integral tan x , \mathrm{d}x = -ln |cos x| + C $

$ integral \cot x , \mathrm{d}x = ln |sin x| + C $

$ integral \sec^2 x , \mathrm{d}x = tan x + C $

$ integral \csc^2 x , \mathrm{d}x = -\cot x + C $

$ integral (\mathrm{d}x)/(sqrt(1 - x^2)) = arcsin x + C $

$ integral (\mathrm{d}x)/(1 + x^2) = arctan x + C $

$ integral \sec x , \mathrm{d}x = ln |\sec x + tan x| + C $

$ integral (1)/(x^2 + a^2) , \mathrm{d}x = (1)/(a) arctan (x)/(a) + C $

$ integral (1)/(sqrt(x^2 plus.minus a^2)) , \mathrm{d}x = ln |x + sqrt(x^2 plus.minus a^2)| + C $

$ integral (1)/(sqrt(a^2 - x^2)) , \mathrm{d}x = arcsin (x)/(a) + C $

$ integral sqrt(x^2 + a^2) , \mathrm{d}x = (x)/(2) sqrt(x^2 + a^2) + (a^2)/(2) ln |x + sqrt(x^2 + a^2)| + C $
